Приложение написано на платформе NET 5.0 / ASP.NET Core WEB API

Тестирование проходило через Swagger.

В программе 3 контроллера обрабатывающие входящие запросы:
 - KrpController
 - CatalogController
 - OrganizationController
 - YandexgeocoderController

1)Получения адреса через сервисы яндекса Yandex Geocoder обрабатывается в контроллере YandexgeocoderController.
 - Метод принимает 2 параметра со строковым типом данных(ключ и адрес)
 - Для отправки запросов используется класс HttpClient. Метод позволяет запрашивать информацию об адресе.


2)Модели представления KRPViewModel и CatalogViewModel для получения и отображения данных с файла.
 - Пути к файлу json с данными справочника деятельности и КРП конфигурируется в KrpController и CatalogController,
   в поле path со строковым типом данных(файлы расположены в папке wwwroot).
   методы десериализует JSON-файлы. Используются модели представления и обычный класс(CatalogService и KRPService)
   где добавлен список данных из модели представления(свойство сопоставляются с ключом и значениям из JSON).
   Для загрузки данных с файла использовался класс WebClient.

3)Имеется модель Organization для CRUD-операций на базе данных MongoDB
 - Строка подключения к базе данных находится в файле конфигураций appsettings.json
 - CRUD-операций реализованы в классе OrganizationService.
 - Через OrganizationController происходит обработка запросов.

4)В OrganizationController, в методе GetAll() идет получение списка организаций, где через зависимость получаем обьект OrganizationService,
  в котором определен метод GetOrganizations(), где из базы данных получаем все обьекты модели организаций.
